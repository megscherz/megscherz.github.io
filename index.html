Actualize Capstone Project

The culmination of my skills learned at Actualize: Alternatively Green. This web application helps users find eco-friendly versions of their favorite household products. The applications leverages a Rails backend framework, PostgreSQL database with several models and joint tables, a VueJS-based frontend as well as a third-party API. I'm excited to continue developing this project and intend to see it through to a live launch.
